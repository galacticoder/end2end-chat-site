# Security Policy

## Supported Versions

I support the latest version of the project only. Please ensure you are using the most recent release to benefit from the latest security patches.

## Reporting a Vulnerability

I take the security of this project seriously. If you have found a vulnerability, please report it to me directly instead of opening a public issue.

### How to Report

Please email me at **galacticoderr@gmail.com**.

In your report, please include:
*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   The version of the software you are running.
*   Any relevant logs or screenshots.

### Response Timeline

I will try to acknowledge your report within 12 hours if i am not busy and at most 48 hours. I will keep you updated on the progress of the fix and let you know when it has been resolved.

## Disclosure Policy

*   Please do not disclose the vulnerability publicly until a fix has been released.
*   I will credit you for your discovery in the release notes (unless you prefer to remain anonymous).

## Scope

### In Scope
*   Any code that is part of this project at all.

### Out of Scope
*   Vulnerabilities in third-party dependencies (unless they are used in an insecure way).
*   Social engineering attacks against users.

Thank you for helping keep this project secure!
